Greetings!

This tool is designed to streamline the creation and design process of analytical dashboards within Excel. This tool automates the generation of the foundational structure for an Excel dashboard.

To get started:

1. Download the complete directory.
2. Right-click on "Main.py" and execute with Python.
3. Input your dashboard parameters in the provided prompt, then click 'run'.
4. Observe the creation of your dashboard in the Excel file named "Dashboard_".

Feel free to anticipate upcoming updates in the near future. Happy dashboard designing!
